# 前言

## 为什么写这本书

量化投资，简单说就是**用数据和程序，把投资决策变成可重复、可检验的规则**。它听起来高深，门槛似乎很高：要懂金融、懂统计、还要会编程。于是很多对量化感兴趣的同学，常常卡在「不知道从哪开始」。

这本书想解决的就是这个问题。我们假设你：

- **没有编程基础**——没关系，第一部分会从 Python 的第一行代码教起；
- **没有金融背景**——没关系，我们会在用到时解释收益率、波动率、因子这些概念；
- **数学只记得一点点**——没关系，需要的概率统计与线性代数会在用到处补足，详见[附录B](appendix/b-math-review.md)。

你需要的，只是愿意一边读、一边把代码敲一遍的耐心。

## 这本书不承诺什么

我们必须先把话说在前面：**这本书不会教你「稳赚不赔」的策略，也不承诺让你「战胜市场」。**

大量研究表明，扣除成本后，绝大多数主动策略长期跑不赢指数。量化投资的真正价值，不在于「精准预测涨跌」，而在于**用系统化、可量化、可复现的方法去发现并管理那些微小但稳定的规律**，同时**避免那些数量惊人的方法论错误**——正是这些错误，让无数策略「回测完美、实盘亏钱」。

学完本书，你应当获得的是一种**能力**：独立地获取数据、构建信号、严谨地回测、诚实地评估，并清楚地知道一个策略为什么可能有效、又为什么可能失效。

## 与《金融数据科学》的关系

本书是《金融数据科学》的**姊妹书**。那本书覆盖面更广，讲的是金融数据科学的方法与工具；本书则更专注、更深入地讲**量化投资这一条流水线**，并且**完全自包含、零基础可读**。两本书有少量基础内容重叠（如 Python、收益率），这是有意为之，让本书可以独立阅读。

## 如何学习

1. **动手**：每章都有配套 notebook，请务必边读边运行，改改参数看看结果会怎么变。
2. **质疑**：看到一条漂亮的净值曲线，先问「它是不是用了未来信息？样本是不是幸存者？」——本书会反复训练这种警惕。
3. **循序**：第一部分打基础，之后按数据→信号→策略→回测→风控的顺序推进，每一部分都建立在前一部分之上。

准备好了吗？我们从认识量化投资开始。
