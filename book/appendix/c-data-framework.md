# 附录C 数据源与框架对比速查

## C.1 A 股数据源对比

| 维度 | AkShare | Tushare | BaoStock |
|---|---|---|---|
| 是否免费 | 完全免费 | 免费+积分制 | 完全免费 |
| 是否需注册 | 否 | 是（需 token） | 否（需 login/logout） |
| 代码写法 | `600519` | `600519.SH` | `sh.600519` |
| 复权参数 | `adjust="qfq/hfq/"` | `pro_bar(adj="qfq")` | `adjustflag="1/2/3"` |
| 覆盖范围 | 股票/期货/基金/宏观/另类，最广 | 股票/财务/指数，规范 | 股票历史，稳定 |
| 返回类型 | DataFrame（中文列名） | DataFrame | 字符串，需转类型 |
| 适合 | 学习、个人研究（本书主线） | 机构、规范字段 | 批量长历史、免费稳定 |

!!! tip "复权口径"
    回测默认用**前复权（qfq）**。三家的复权口径细节略有差异，跨源拼接数据时务必统一为同一口径与同一时间基准。

## C.2 回测框架对比

| 维度 | akquant（主线） | Backtrader | vectorbt | Qlib |
|---|---|---|---|---|
| 范式 | 事件驱动 | 事件驱动 | 向量化 | 工作流/平台 |
| 内核 | Rust + Python | 纯 Python | NumPy 向量化 | Python + C 扩展 |
| 速度 | 快 | 一般 | 很快 | 取决于流程 |
| 上手难度 | 中 | 低（文档全） | 中高（API 陡） | 高 |
| 撮合精细度 | 高 | 高 | 中 | 高 |
| 强项 | akshare 生态、滚动验证、报告 | 教学、指标丰富 | 海量参数扫描 | ML 选股全流程 |
| 适合 | 本书主线、贴近真实 | 入门教学 | 信号/参数筛选 | 机构 ML 流水线 |

选择建议：

- 快速研究、参数扫描 → **vectorbt**；
- 贴近真实撮合、加风控 → **akquant / Backtrader**；
- 机器学习选股全流程 → **Qlib**。
