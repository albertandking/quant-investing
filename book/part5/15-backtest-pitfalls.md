## 15.1 本章导读：回测是实验室，也是照妖镜

回测，就是**用历史数据模拟「如果当时这样交易，结果会怎样」**。它是量化研究的实验室——没有它，策略只是空想。

但回测也极其容易**自欺**。同一个策略，写法上一个不起眼的差错，就能让年化收益凭空多出好几个百分点；一组参数试了上百遍挑出最好的那个，回测曲线漂亮得像艺术品，实盘却一上线就亏。本章先讲**正确的向量化回测怎么写**，再系统拆解四个最常见、也最致命的陷阱：

- **前视偏差**：用了当时根本拿不到的未来信息；
- **交易成本**：忽略它，高换手策略的「收益」多半是假的；
- **幸存者偏差**：只用「活到今天」的股票做样本；
- **过拟合 / 数据窥探**：在历史上反复挑参数，挑出的是运气不是规律。

学完本章，你看任何一条净值曲线，第一反应都会是「它是不是哪里作弊了？」——这正是一个量化研究者最该有的本能。

## 15.3 向量化回测：最朴素也最常用的写法

「向量化回测」的思路是：先把每天**该不该持仓**算成一个 0/1（或权重）序列 `position`，再用它乘以每天的收益率，一次性得到策略每天的收益。它不模拟逐笔下单，所以快，适合快速筛选信号与扫参数。

以「5 日均线上穿 20 日均线就持有，否则空仓」为例：

```python
from qfin import load_close, sma, sharpe_ratio, annualized_return

px = load_close()["STK001"]
ret = px.pct_change().fillna(0)                 # 每日收益率

signal = (sma(px, 5) > sma(px, 20)).astype(int)  # 1=看多, 0=空仓
position = signal.shift(1).fillna(0)             # 关键：信号滞后一天再成交
strat = position * ret                           # 策略每日收益

print(f"年化收益：{annualized_return(strat):.2%}，夏普：{sharpe_ratio(strat):.2f}")
# 在 STK001 上：年化 19.48%，夏普 1.43
```

那行 `signal.shift(1)` 是整个回测正确性的**地基**。

!!! warning "T+1 与前视：为什么必须 shift(1)"
    `signal` 由**当日收盘价**算出——也就是说，你要到**今天收盘那一刻**才知道今天该不该买。可那时已经收盘了，你最早只能**明天**才能按价格成交；何况 A 股 T+1，今天买的明天才能卖。所以「今天的信号」必须对应「明天的收益」，即 `position = signal.shift(1)`。

    漏掉这个 `shift(1)`，等于偷看了未来——这就是下一节要量化的**前视偏差**。

## 15.4 前视偏差：最隐蔽的作弊

**前视偏差（Look-ahead Bias）**：回测中使用了在决策时点根本无法获得的信息。它往往藏在一行代码里，却能把一个平庸策略包装成「圣杯」。

把上面正确的写法（`signal.shift(1)`）和错误的写法（`signal` 当天就成交）对比一下：

```python
strat_correct = signal.shift(1).fillna(0) * ret    # 正确：T+1
strat_lookahead = signal * ret                      # 错误：当天信号当天成交
```

在 `STK001` 上实测：

| 写法 | 年化收益 | 夏普比率 |
|---|---|---|
| 正确（`shift(1)`） | 19.48% | 1.43 |
| 前视（当天成交） | 27.26% | 1.94 |
| **差额（前视溢价）** | **+7.78%** | +0.51 |

!!! example "例 15.1　凭空多出的 7.78%"
    仅仅因为「让信号当天就成交」，年化收益就从 19.48% 跳到 27.26%——**7.78 个百分点的「超额收益」完全是假的**，来自偷看了今天收盘才知道的信息。实盘里你不可能用今天的收盘价信号去成交今天，这部分收益一分钱都拿不到。

前视偏差还有很多更隐蔽的变体：用了「未来才公布」的财务数据（财报有披露滞后）、用全样本的均值/标准差去标准化历史数据、用复权因子时引入了未来的除权信息……第7章已强调过数据层面的防范，**回测层面就靠 `shift` 把一切信号、特征都严格滞后到可获得的时点**。

## 15.5 交易成本、换手与滑点

回测里另一个常见的「美化」，是假装交易不要钱。真实交易有：

- **佣金**：券商收取，通常万几到万三（双边）；
- **印花税**：卖出时 0.05%（2023 年起，单边）；
- **滑点**：实际成交价与你看到的价格之间的偏差，尤其在流动性差或下单量大时。

把这些合并成一个**单边成本率**，每次调仓（换手）按换手金额扣减。换手率用持仓的变化来度量：

```python
turnover = position.diff().abs().fillna(0)   # 每天的换手（0→1 或 1→0）
cost_rate = 0.001                            # 单边千分之一（含佣金/印花/滑点的粗略合计）
strat_net = strat - turnover * cost_rate
```

在 `STK001` 上，这个均线策略**年均单边换手约 13.9 次**，不同成本下的结果：

| 单边成本 | 年化收益 | 夏普 |
|---|---|---|
| 0.00% | 19.48% | 1.43 |
| 0.05% | 18.65% | 1.37 |
| 0.10% | 17.83% | 1.32 |
| 0.20% | 16.21% | 1.21 |

!!! tip "换手越高，对成本越敏感"
    这个策略换手不算高，成本侵蚀还温和。但一个**日频高换手**策略，年换手可能上百次，0.1% 的成本就足以把账面上漂亮的收益吃光。**评估任何策略，都要在加入合理成本后再看绩效**，而不是只看零成本的理想曲线。

## 15.6 幸存者偏差

**幸存者偏差（Survivorship Bias）**：样本里只包含「活到今天」的标的，那些退市、被并购、长期停牌的股票被悄悄剔除了。

设想你用「**当前**沪深300成分股」回测过去十年的策略：这些公司能留在指数里，本身就说明它们过去十年表现不差——你等于提前知道了「赢家名单」。那些十年间暴雷退市的公司从未进入你的样本，回测收益自然被系统性高估。

!!! warning "本书内置数据的提醒"
    本书内置的 40 只合成股票**没有退市**，因此天然不含幸存者偏差——但这正是「理想化」的地方。**用真实数据时**，务必使用**含退市股的「时点成分股（point-in-time）」数据**：在每个历史时刻，用的是当时真实存在、真实在册的股票池，而不是今天的名单。

## 15.7 过拟合与数据窥探

这是最难防、也最致命的陷阱。

**数据窥探（Data Snooping）/ 过拟合（Overfitting）**：在同一段历史数据上反复尝试很多策略或参数，挑出表现最好的那个。问题在于——**只要试得够多，纯粹靠运气也能找到一条漂亮的曲线**。

下面这个实验最能说明问题。我们生成 100 个**完全随机**的买卖信号（和价格毫无关系），各自回测，取其中夏普最高的：

```python
import numpy as np
from qfin import sharpe_ratio

rng = np.random.default_rng(0)
best = max(sharpe_ratio(rng.integers(0, 2, len(ret)) * ret) for _ in range(100))
print(f"100 个随机策略中最优夏普：{best:.2f}")   # 结果：2.04
```

!!! example "例 15.2　随机的胜过真实的"
    我们精心设计的均线策略，夏普是 **1.43**；而 100 个**纯随机**信号里，最幸运的那个夏普高达 **2.04**——比真策略还高！如果你试了 100 组参数只汇报最好的一组，你汇报的很可能就是这种「运气选手」。试的次数越多，这个虚高的「最优夏普」越离谱。这就是为什么**「我调出了一个夏普 2 的策略」本身毫无意义**——关键是你试了多少次、它在没见过的数据上还成不成立。

### 如何防范：稳健性检验

1. **参数高原 > 参数孤峰**：好策略在一**片**参数附近都表现稳定，而不是只有某一个孤立参数点好。看一下窗口扫描：

    | 参数 (短, 长) | 夏普 |
    |---|---|
    | (5, 20) | 1.43 |
    | (10, 30) | 1.38 |
    | (10, 60) | 1.52 |
    | (20, 60) | 1.67 |

    这几组参数夏普都在 1.4–1.7 之间，相对平稳，是个不错的信号；如果只有某一个参数点夏普飙到 3、邻近全是负的，那几乎肯定是过拟合。

2. **样本外检验**：把数据按时间切成「样本内（前 70%）」和「样本外（后 30%）」。只在样本内调参，到样本外验证。样本外大幅衰减，说明样本内是过拟合。

3. **控制尝试次数**：诚实记录你试了多少组参数。试得越多，对「最优结果」越要打折。

!!! warning "回测的真正目的"
    回测**不是**为了证明策略一定有效，而是为了尽早发现它**为什么可能无效**。一个量化研究者的价值，不在于画出漂亮的净值曲线，而在于对自己的策略足够「狠」——主动地去否证它。

## 15.8 本章小结

**必须掌握**

1. **时序正确性是地基**：`position = signal.shift(1)` 对应 A 股 T+1，漏掉它就是前视偏差。在 `STK001` 上前视让年化凭空多出 7.78 个百分点。
2. **成本不可省略**：双边佣金、印花税、滑点都要算进去，高换手策略尤其敏感；评估绩效务必看「加成本后」的结果。
3. **警惕过拟合**：100 个随机策略的最优夏普能到 2.04，远超真实策略——单看「最优结果」毫无意义。

**理解即可**

4. 幸存者偏差：真实回测要用含退市股的时点成分股数据。
5. 稳健性检验：参数高原优于孤峰，样本外优于样本内最优。

**实践提醒**

养成「看到漂亮曲线先怀疑」的本能：它是不是 shift 漏了？算成本了吗？样本是不是只有幸存者？参数是不是试了一万遍？
