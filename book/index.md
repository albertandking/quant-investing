# 量化投资

> 一本面向中国本科生、研究生与量化投资爱好者的**开源、免费、零基础**量化投资实战教程。

## 本书定位

市面上的量化投资资料，要么是面向从业者的高门槛专著，要么是零散的博客与代码片段。本书想做的，是一份**系统、自包含、可动手**的中文教程：

- **零编程基础起步**：从 Python 语法、NumPy/Pandas 讲起，不要求任何编程或金融背景，只需要高中数学与一点好奇心。
- **贴近中国市场**：示例与案例尽量采用 **A 股**及中国市场数据，正视 T+1、涨跌停、停牌等制度特征。
- **完整流水线**：沿着「**数据 → 信号 → 因子 → 策略 → 组合 → 回测 → 风控 → 实盘**」这条主线，把零散的知识点串成可执行的工作流。
- **真实可跑的代码**：所有代码都能在本地或云端真正运行，配套内置数据集**断网也能跑**。

!!! warning "重要风险提示"
    本书所有内容仅用于**教育与研究目的，不构成任何投资建议**。书中内置数据多为合成示例，策略示例不代表真实收益。**量化投资有风险，入市需谨慎。**

## 本书特色

- **正文 + 代码分离**：概念与推导写在 Markdown 正文，可运行代码放在配套 Jupyter Notebook，正文按需引用代码与图表。
- **本地可执行**：所有基础章节依赖**内置示例数据集**，断网也能跑；进阶章节提供**中国市场真实数据**接口（akshare / tushare / baostock）。
- **主线统一、多框架对比**：回测主线用事件驱动框架 **akquant**，并在标签页中对比 **Backtrader / vectorbt / Qlib**，方便读者迁移到自己熟悉的工具。

## 如何使用本书

1. 按[附录A](appendix/a-setup.md)配置本地环境（`uv sync` 一条命令）。
2. 生成内置数据：`uv run python scripts/make_sample_data.py`。
3. 阅读正文，配合 `notebooks/` 下对应章节的配套 notebook 逐步运行。

完全的零基础读者，建议从第一部分按顺序学习；有 Python 与金融基础的读者，可直接跳到感兴趣的部分。

## 全书结构

| 部分 | 主题 | 章节 |
|---|---|---|
| 第一部分 | 入门与基础 | 第1–4章：导论、Python、NumPy/Pandas、金融与统计基础 |
| 第二部分 | 数据 | 第5–7章：行情获取、财务与另类数据、预处理与特征 |
| 第三部分 | 信号与因子 | 第8–10章：技术信号、单因子检验、多因子模型 |
| 第四部分 | 策略 | 第11–14章：趋势/均值回归、配对交易、多因子选股、资产配置 |
| 第五部分 | 回测 | 第15–17章：回测陷阱、akquant 事件驱动回测、绩效与归因 |
| 第六部分 | 风险与实盘 | 第18–20章：风险与仓位、机器学习、从回测到实盘 |
| 附录 | 参考 | 环境配置、数学回顾、数据源/框架对比、术语表、资源清单 |
